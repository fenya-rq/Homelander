# 🥗 Homelander Nutrition Bot

**AI-ассистент для контроля питания в Telegram.** Забудь про ручной поиск продуктов в базах данных и самостоятельное высчитывание граммов. Просто напиши боту, что ты съел, а он сам определит КБЖУ с помощью LLM и сохранит это в твою личную историю.

---

✨ Основные функции

* **Natural Language Processing:** Понимает сложные фразы на живом языке, например: "Съел два яйца вкрутую, чашку кофе с молоком и две дольки темного шоколада".
* **Мультимодальный ввод (Voice-to-Text):** Интеграция с ElevenLabs API позволяет отправлять голосовые сообщения — бот мгновенно транскрибирует их и обрабатывает как текст.
* **Автоматический расчет КБЖУ:** Использует Google Gemini для извлечения калорий, белков, жиров, углеводов и клетчатки (PFCF) даже из примерных описаний порций.
* **Интеллектуальные уточнения:** Если данных недостаточно для точного расчета, бот задаст уточняющий вопрос вместо того, чтобы записывать «пустые» данные.
* **Гибкая аналитика:** Команды статистики выводит агрегированные итоги за сутки и неделю с корректным учетом часовых поясов.
* **Stateless Архитектура:** Надежная обработка конкурентных запросов в асинхронном режиме — данные пользователей изолированы и никогда не перемешаются.
* **Modern Tech Stack:** Сборка на базе aiogram 3.x, менеджер пакетов uv и контейнеризация через Docker для быстрого развертывания.

---

## 🛠 Технологический стек

* **Core:** `Python 3.14+`
* **Bot Framework:** [aiogram 3.x](https://github.com/aiogram/aiogram) — современный асинхронный фреймворк.
* **AI Engine:** [Google Gemini API](https://ai.google.dev/) — модели `Gemini` для мгновенного парсинга состава продуктов.
* **Voice-to-Text:** [ElevenLabs](https://elevenlabs.io/) — высокоточное распознавание речи (STT) для обработки голосовых сообщений.
* **Database:** `SQLite` + [aiosqlite](https://github.com/omnilib/aiosqlite) — асинхронная работа с локальной БД.
* **Data Validation:** [Pydantic v2](https://docs.pydantic.dev/) — использование DTO для строгого контроля типов и структуры данных.
* **Architecture:** `Stateless Manager` (синглтон) для безопасной работы в `asyncio` среде.
